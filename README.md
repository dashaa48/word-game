# Word Game (Игра в слова)

Консольная игра в слова на C++17 с поддержкой нескольких режимов игры.

## Описание

Игра, в которой игроки по очереди называют слова на последнюю букву предыдущего слова. Программа контролирует:
- Правильность первой буквы;
- Наличие слова в словаре;
- Отсутствие повторов.

## Возможности

- **3 режима игры:**
  - Человек против компьютера;
  - Два человека;
  - Два человека против компьютера.
- **Умный словарь:** слова загружаются из текстовых файлов (по файлу на букву a-z).
- **Компьютер:** компьютер выбирает случайные слова из словаря.
- **Fallback-логика:** если на текущую букву нет слов, игра ищет предыдущую букву в этом слове.
- **Валидация ввода:** проверка, что слово состоит только из букв.

## Технологии

- **Язык:** C++17

- **Тестирование:** doctest
- **Документация:** Doxygen

- `src/utils.cpp` — вспомогательные функции;
- `tests/test_game.cpp` — модульные тесты (doctest);

- `.clang-format` — настройки форматирования кода.

## Как собрать и запустить

### Требования

- CMake 3.14+;
- Компилятор с поддержкой C++17.

### Сборка

```bash
cmake -B build -S .
cmake --build build --config Release
```

## Запуск игры

На Linux/macOS:

```bash
./build/word_game
```

На Windows (PowerShell):

```powershell
.\build\Release\word_game.exe
```

## Запуск тестов

### После сборки

```bash
cd build
ctest --output-on-failure
```

Ожидаемый результат:
`100% tests passed, 0 tests failed out of 1`

## Документация

## Правила игры
- Игра начинается с буквы 'a';
- Игрок должен назвать слово на текущую букву.

### Слово должно
- Начинаться с требуемой буквы;
- Быть в словаре;
- Не быть использованным ранее;
- Состоять только из букв;
- Следующая буква - последняя буква названного слова;
- Если на текущую букву нет слов, игра ищет fallback-букву в предыдущем слове;
- Игра заканчивается, когда игрок не может назвать слово.
